# 企业财务管理系统 java416

## 项目概述

企业财务管理系统是基于 Spring Boot + Vue 的前后端分离架构，旨在为各类企业提供一套完整的财务解决方案。本系统具备完善的权限管理，确保数据安全，同时提供友好的用户界面，提高工作效率。

## 技术栈

- **后端：** Java + Spring Boot + MyBatis + Maven
- **前端：** Vue.js
- **数据库：** MySQL 5.7
- **开发工具：** IntelliJ IDEA + Visual Studio Code
- **运行环境：** JDK 1.8 + Node.js 14 版本

## 功能模块

- **系统管理员：** 管理系统设置、用户权限分配等
- **财务人员管理：** 负责财务数据的录入、审核等
- **员工模块：** 员工信息管理、财务申请等

## 系统角色

- 系统管理员：负责系统的维护与管理
- 财务人员：处理财务相关的业务
- 员工：进行财务申请等操作

## 环境配置

- **JDK：** 1.8 版本
- **Maven：** 用于项目构建和依赖管理
- **MySQL：** 5.7 版本
- **Node.js：** 14 版本

## 部署步骤

1. 环境配置，确保上述依赖已正确安装
2. 数据库初始化，执行 SQL 脚本建立数据库及表结构
3. 后端编译，使用 Maven 进行编译打包
4. 前端编译，使用 npm 或 yarn 进行编译打包
5. 部署应用到服务器

## 目录结构

```
项目根目录
├── backend                 # 后端代码目录
├── frontend                # 前端代码目录
└── database                # 数据库脚本目录
    ├── schema.sql          # 数据库结构脚本
    └── data.sql            # 数据库数据初始化脚本
```
